# Financial Reporting & KPI Analysis Dashboard

## Project Overview
This project presents a financial performance dashboard built using Power BI to analyze revenue, profit margins, operating expenses, and sales performance across different countries.

## Tools Used
- Power BI
- Excel
- Data Cleaning
- Financial KPI Analysis

## Dashboard Preview
![Financial Dashboard](financial-dashboard.png.png)

## Key Insights
- Revenue trends from 2018–2020
- Gross profit and net profit margins
- Country-wise sales performance
- Sales vs Marketing cost comparison
